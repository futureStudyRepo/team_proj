# TRA06 인수시험 시나리오 예시

문서명: 인수시험 시나리오

## 1. 시나리오

| 시나리오 ID | 내용 | 승인 주체 |
|---|---|---|
| AT-01 | 당뇨 환자 차트 조회 시 HbA1c 누락 배너 확인 | 내분비내과 |
| AT-02 | 외부검사 결과 등록 시 배너 미표시 확인 | 내분비내과 |
| AT-03 | 응급실 활력징후 입력 후 패혈증 고위험 팝업 확인 | 응급의학과 |
| AT-04 | 패혈증 경고 후 후속조치 로그 기록 확인 | 감염관리실 |
| AT-05 | 운영자 화면 KPI 조회 확인 | 의료정보팀 |

## 2. 합격 기준

- 핵심 시나리오 전부 성공
- 중대 결함 0건
- 임상 부서 승인

## 3. 한 줄 정리

인수시험 시나리오의 핵심은  
`최종 사용자가 실제 업무 기준으로 받아들일 수 있는지 확인하는 것`이다.
