# TES02 시스템시험 결과서 예시

문서명: 시스템시험 결과서

## 1. 시험 결과 요약

| 시험 ID | 시나리오 | 결과 | 비고 |
|---|---|---|---|
| ST-01 | 외래 진료 중 HbA1c 누락 확인 | 성공 | 배너/상세보기 정상 |
| ST-02 | 안저검사 예약 연계 | 성공 | 예약지원 화면 이동 확인 |
| ST-03 | 응급실 패혈증 고위험 경고 | 성공 | 경고 팝업 5초 이내 표시 |
| ST-04 | 경고 후 오더 반영 | 성공 | 로그 저장 정상 |
| ST-05 | 운영자 KPI 조회 | 성공 | 기본 통계 조회 가능 |

## 2. 사용자 피드백

| 사용자 | 피드백 | 반영 여부 |
|---|---|---|
| 외래 의사 | 배너 문구는 짧을수록 좋다 | 반영 예정 |
| 응급실 의사 | 패혈증 팝업에 수치가 바로 보여야 한다 | 반영 완료 |

## 3. 한 줄 정리

시스템시험 결과서의 핵심은  
`실제 사용자 입장에서 기능이 자연스럽게 동작하고 수용 가능한지 확인하는 것`이다.
