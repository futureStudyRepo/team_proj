# DES11 통합시험 시나리오 예시

문서명: 통합시험 시나리오  
대상 기능:

- 당뇨 추적검사 누락 알림
- 패혈증 위험 알림

## 1. 시나리오 목록

| 시나리오 ID | 시나리오명 | 연계 대상 | 기대 결과 |
|---|---|---|---|
| IT-01 | 환자 차트 조회 시 당뇨 알림 판정 | HIS + EMR + LIS + CDSS | HbA1c 누락 배너 표시 |
| IT-02 | 검사결과 수신 후 당뇨 알림 종료 | LIS + CDSS + HIS UI | 알림 상태 자동 종료 |
| IT-03 | 활력징후 저장 후 패혈증 판정 | NURSING + CDSS + HIS UI | qSOFA 고위험 팝업 표시 |
| IT-04 | 패혈증 경고 후 오더 연계 | OCS + CDSS + 로그 | 후속조치 기록 저장 |

## 2. 대표 시나리오 예시

### IT-01

1. 외래 환자 차트를 연다.
2. 시스템이 진단코드와 당뇨약 이력을 조회한다.
3. LIS에서 최근 HbA1c 결과일을 조회한다.
4. 마지막 검사일이 90일 초과이면 배너를 표시한다.

### IT-03

1. 응급실에서 활력징후를 저장한다.
2. 시스템이 SBP, RR, 의식상태를 읽는다.
3. qSOFA 점수를 계산한다.
4. 2점 이상이면 고위험 팝업을 표시한다.

## 3. 한 줄 정리

통합시험의 핵심은  
`각 시스템 간 데이터 흐름이 실제로 끊김 없이 연결되는지 검증하는 것`이다.
