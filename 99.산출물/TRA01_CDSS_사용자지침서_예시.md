# TRA01 사용자 지침서 예시

문서명: 사용자 지침서  
대상 사용자:

- 주치의
- 응급실 의사
- 간호사
- 코디네이터

## 1. 당뇨 추적검사 누락 알림 사용법

1. 외래 진료화면 상단 배너를 확인합니다.
2. `상세보기`를 눌러 마지막 검사일과 발생 근거를 확인합니다.
3. 필요 시 `검사 오더`를 눌러 HbA1c 검사를 처방합니다.
4. 즉시 조치하지 않을 경우 `무시`를 눌러 사유를 기록합니다.

## 2. 패혈증 고위험 알림 사용법

1. 응급실 또는 병동에서 경고 팝업이 뜨면 즉시 확인합니다.
2. 팝업 본문에서 수축기혈압, 호흡수, 의식상태 등 발생 근거를 확인합니다.
3. 상세화면에서 혈액배양, 젖산, 항생제 검토 항목을 확인합니다.
4. 필요한 처방을 입력하고 환자 상태를 재평가합니다.

## 3. 주의사항

- 알림은 의사결정을 보조하는 기능이며, 최종 판단은 의료진이 수행합니다.
- 반복 경고가 과도하다고 판단되면 운영부서에 피드백을 전달합니다.
- 무시 사유는 운영성과 분석에 사용되므로 가능한 구체적으로 입력합니다.

## 4. 한 줄 정리

사용자 지침서의 핵심은  
`알림을 단순히 확인하는 데서 끝내지 않고, 적절한 후속조치와 기록으로 연결하는 것`이다.
