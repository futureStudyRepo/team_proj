# ANA01 업무정의서 예시

문서명: 업무정의서  
대상 시스템: 병원행정시스템(HIS) 연계 CDSS  
대상 기능:

- 당뇨 환자 추적검사 누락 알림
- 패혈증 위험 알림

## 1. 문서 목적

본 문서는 CDSS 도입 전후의 업무 흐름을 정의하고, 사용자 역할과 업무 책임, 개선 포인트를 명확히 하기 위한 것이다.

## 2. 대상 업무

| 업무영역 | 업무명 | 주요 사용자 |
|---|---|---|
| 외래 | 당뇨 지속관리 | 주치의, 외래 간호사, 코디네이터 |
| 응급실 | 패혈증 조기 인지 및 대응 | 응급실 의사, 간호사 |
| 병동 | 패혈증 재평가 및 추적 | 병동 의사, 간호사 |
| 운영 | 규칙 관리 및 성과 모니터링 | 의료정보팀, 감염관리실 |

## 3. AS-IS 업무 흐름

### 3.1 당뇨 추적검사

1. 환자 차트 조회
2. 의료진이 과거 검사 결과를 직접 확인
3. 필요 시 검사 처방 또는 예약 연계
4. 누락 여부는 개인 경험에 따라 판단

문제점:

- 의료진이 수동으로 확인해야 하므로 누락 가능성이 높다.
- 외부검사 결과 반영 여부가 일관되지 않다.

### 3.2 패혈증 위험 인지

1. 간호사가 활력징후를 입력
2. 담당자가 환자 상태를 종합 판단
3. 필요 시 의사 보고 및 추가 처치

문제점:

- 활력징후와 의식상태, 검사결과를 동시에 보기 어렵다.
- 초기 대응이 경험 의존적일 수 있다.

## 4. TO-BE 업무 흐름

### 4.1 당뇨 추적검사

1. 환자 차트 조회
2. 시스템이 당뇨 관리대상 여부를 자동 식별
3. HbA1c/안저검사 누락 여부 자동 판정
4. 진료화면 배너 표시
5. 사용자가 검사 오더 또는 예약 연계
6. 처리 결과 로그 저장

### 4.2 패혈증 위험 인지

1. 활력징후 입력
2. 시스템이 qSOFA 자동 계산
3. 고위험 시 즉시 팝업 경고
4. 사용자가 상세정보 확인 후 추가 조치
5. 후속조치 로그 저장

## 5. 역할 정의

| 역할 | 책임 |
|---|---|
| 주치의 | 당뇨 알림 확인, 검사 처방, 진료계획 반영 |
| 코디네이터 | 안저검사 예약 연계, 외부검사 결과 등록 보조 |
| 응급실 의사 | 패혈증 고위험 경고 확인 및 즉시 조치 |
| 간호사 | 활력징후 정확 입력, 재평가 |
| 감염관리실 | 조치율/KPI 분석, 운영 정책 검토 |
| 의료정보팀 | 규칙 운영, 인터페이스 점검, 장애 대응 |

## 6. 한 줄 정리

업무정의서의 핵심은  
`CDSS가 들어오면서 누가 어떤 판단과 조치를 더 빠르고 일관되게 하게 되는지 업무 흐름 관점에서 보여주는 것`이다.
