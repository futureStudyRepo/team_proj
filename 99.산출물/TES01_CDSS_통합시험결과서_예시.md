# TES01 통합시험 결과서 예시

문서명: 통합시험 결과서

## 1. 시험 결과 요약

| 시험 ID | 항목 | 결과 | 비고 |
|---|---|---|---|
| IT-01 | 환자 차트 조회 시 당뇨 알림 판정 | 성공 | HbA1c 누락 배너 정상 표시 |
| IT-02 | 검사결과 수신 후 알림 종료 | 성공 | LIS 결과 반영 후 알림 종료 |
| IT-03 | 활력징후 저장 후 qSOFA 판정 | 성공 | 5초 이내 고위험 팝업 표시 |
| IT-04 | 패혈증 경고 후 오더 연계 | 부분성공 | 일부 로그 코드명 정리 필요 |

## 2. 주요 결함

| 결함 ID | 내용 | 조치 |
|---|---|---|
| BUG-IT-01 | 후속조치 로그 코드명이 운영 표준과 불일치 | 코드셋 수정 예정 |

## 3. 한 줄 정리

통합시험 결과서의 핵심은  
`시스템 간 연결은 되었는지, 그리고 연결된 결과가 기대한 업무 흐름으로 이어지는지 확인하는 것`이다.
